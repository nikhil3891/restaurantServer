# API Documentation

REST API contracts, endpoint references, request/response schemas, and integration guides.

## What goes here
- API endpoint reference sheets (per module)
- Request and response payload examples
- Authentication and authorization flow docs
- Error codes and error response formats
- Rate limiting and throttling policies
- Third-party API integration notes (Swiggy, Zomato, Payment gateways)
- Swagger/OpenAPI spec files
- Versioning policy (`/api/v1/`, `/api/v2/`)
- Webhook documentation
