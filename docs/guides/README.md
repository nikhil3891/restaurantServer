- Developer onboarding guide (how to run the project locally)
- Coding standards and conventions (naming, folder structure)
- Git branching strategy and commit message format
- Code review guidelines
- Testing strategy (unit, integration, e2e)
- Logging and error handling standards
- Security best practices for contributors
- Module creation guide (how to add a new NestJS module)
- Environment setup guide
