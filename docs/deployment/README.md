- Local development setup guide (Docker Compose)
- Production deployment guide
- Environment variables reference (.env keys and descriptions)
- Docker and Dockerfile documentation
- Kubernetes manifests and configuration notes
- AWS infrastructure setup (EC2, RDS, S3, ElastiCache)
- NGINX configuration notes
- CI/CD pipeline documentation (GitHub Actions)
- Release and rollback procedures
- Monitoring and alerting setup (Prometheus, Grafana)
